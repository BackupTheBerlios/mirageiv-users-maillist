... actually, not "exactly", I just miss one customization.
I want to be able to exit Mirage just by pressing "Escape" key.
So I made some modifications to do that, (maybe it could be useful for
someone else...).

Let explain myself :

First problem : 'Escape' key cannot be used for 'custom actions' because
it's reserved for 'Exit Full Screen', but it is also possible to exit
fullscreen by using 'F11' key.
So problem easily solved : I removed this 'Escape' shortcut so it is now
available for use in 'custom actions'

Second problem : what command to use for exiting mirage when I want to
define my custom action?
I could use "killall mirage" but it's really a dirty solution!
So I created an '[EXIT]' operation taking example on '[PREV]' and '[NEXT]'
operations

I can now create a custom action to quit mirage using Escape key :)

But you will say me "and what about old mirage users that are used to use
Escape to exit fullscreen?'
Ok, just define a custom action to do this!
Crap, there's no 'operation' to do this...
so I also created '[EXIT_FULLSCREEN]' operation so we can now create a
custom action to exit fullscreen
And I created it as default like 'open in GIMP' for example, so there's no
visible change to the default built application.
