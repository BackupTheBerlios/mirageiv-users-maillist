On 8/8/06, Thomas Güttler <hv at tbz-pariv.de> wrote:
> Hi,
>
> I patched mirage.py, so that you can define keyborard
> shortcuts
> on the command line:
>
> There are two new command parameter:
> -c, --command        'k: mycommand IMAGE' Execute
>                     'mycommand' if you hit key 'k'"
>                    'k' can be any ASCII character.
>                    You can give several -c options
>                   IMAGE gets replaced with the current
>                    filename
>                  If the command contains the string
>                  NEXT, the next
>                 image gets loaded after the command
>
> -o, --onload         'mycommand IMAGE'
>         Execute 'mycommand',when the image gets loaded
