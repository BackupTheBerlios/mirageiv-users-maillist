Hi all,

I've implemented a patch that adds a "-n/--no-sort" command line
option flag to disable the automatic input parameter sorting that
Mirage does by default. I find it useful to specify a certain sort
order sometimes, and a simple way to implement that is to just sort
the filenames on the command line before passing them in to Mirage.

Please consider the attached patch for inclusion.
