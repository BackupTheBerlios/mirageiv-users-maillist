Hi. I'm Daniel Martín Yerga.

I made a Mirage's port for the Internet Tablets. Internet Tablets are a
few devices that use a modified version of Debian Linux. These devices
are the Nokia 770[1] and the Nokia N800[2].

They use a library that is called Hildon[3], which is GTK modified to
fix on the small screens (800x480), and also GTK himself. I decided to
port Mirage since it's a fast application and with a great potential,
and I use it enough.

I have modified Mirage in order that it use the Hildon library and
specific others of the Internet Tablets. Also I modified some dialogs
in order that they were seen well on these screens. The source code is
in https://garage.maemo.org/projects/mirage/
I might made a patch if Scott believes it suitably, though for the
Desktop version would not be very useful.
You can see some screenshots in my Flickr account:
http://www.flickr.com/photos/yerga/tags/mirage/

Well, for any doubt or suggestion, please to comment me about it.

Sorry for my bad English :(
