Mirage 0.7.1 Released

This is mainly a bug fix release. Note that this version also
introduces an optional dependency, gnome-python, if you want Mirage's
toolbar to follow your gnome preference.

CHANGELOG

- Toolbar style follows gnome preference (requires optional
gnome-python dependency) (Eugenia Loli-Queru)
- Skip hidden files/dirs when loading images
- Bug: Caps-lock breaks some shortcuts (Abisser T'khin)
- Bug fixes
